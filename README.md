# Chess Project

A simple chess application built with TypeScript, HTML, and CSS.

## Getting Started

```bash
git clone https://github.com/<your-username>/chess_project.git
cd chess-project
npm install
npm run dev
```

Visit http://localhost:3000 to view the application.
